# Code of Conduct

This project follows the
[Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, no harassment. Maintainers may edit
or remove contributions, and may ban contributors, for behaviour that violates
this code.

Report unacceptable behaviour to **support@chu6a.dev**. Reports are handled
confidentially.
